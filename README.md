### Comandos útiles

Iniciar MCP server:
```bash
fastmcp run <path_to_server>
fastmcp run <path_to_server> --transport http --port 8080
```

Inspeccionar MCP servers:
```bash
fastmcp dev inspector <path_to_server>
```
